-> 1 byte = 8 bits = 2 hexa
-> 1 endereço de memória armazena 1 byte
	|-> int => 4 bytes
	|
	|-> float => 4 bytes
	|
	|-> Double => Long float => 8 bytes
	|
	|-> char => 1 bytes 

*Nota: tem um de bit extra que representa o sinal do valor, no caso dos tipos númericos; perguntar pro carlos.

-> Por uma questão de condensação de algarismos, usa-se exadecimal pra representar endereços de memória, 
   cada 1 hexa de endereço é um byte
	|-> 1>2>3>4>5>6>7>8>9>0>A>B>C>D>E>F
